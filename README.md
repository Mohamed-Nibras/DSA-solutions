# DSA Solutions

A curated collection of my Data Structures and Algorithms solutions solved through consistent problem-solving practice.

This repository documents my long-term journey of strengthening algorithmic thinking, problem-solving ability, and interview-level programming skills as part of my Software Engineering and AI learning path.

The emphasis is on understanding patterns, writing clean implementations, and developing efficient solutions rather than simply increasing the number of solved problems.

## Repository Structure

```text
DSA-solutions/
│
├── Easy/
├── Medium/
├── Hard/
│
├── README.md
├── .gitignore
└── .gitattributes
```

Problems are organized by their difficulty.

Each solution follows the naming convention:

```text
<Problem Number> - <Problem Name>.py
```

Example:

```text
0001 - Two Sum.py
0088 - Merge Sorted Array.py
0242 - Valid Anagram.py
```
